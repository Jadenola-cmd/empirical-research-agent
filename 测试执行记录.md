异常测试结论：

- 业务错误多数能返回 JSON。
- `json.loads` 发生在 try/except 外，非法 config 会暴露 Python traceback。
- PSM/DID 返回 `{error: ...}` 时退出码仍为 0，不利于自动化测试和上层 Agent 判断失败。
