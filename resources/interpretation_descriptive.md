# 解读检查清单：描述统计 / 相关分析

适用于 `run_analysis.py` 的 `descriptive` 与 `correlation` 方法输出。

## 1. 系数符号与方向是否符合理论预期
- 相关分析给出的是**双变量**关联方向，不能直接当作因果方向或排除混杂因素后的净效应；写作时用"X与Y呈正/负相关"而非"X导致Y上升/下降"。
- 若某两个变量的相关系数符号与理论预期相反，先检查是否存在反向因果或遗漏的第三变量，不要直接断言"假设不成立"——先在OLS/面板回归中控制其他变量后复核。

## 2. 显著性阈值如何表述
- 描述统计本身没有p值，不涉及显著性表述；相关系数的显著性按 `***p<0.01, **p<0.05, *p<0.1` 标注，论文正文表述为"X与Y在5%水平上显著正相关（r=0.xx, p<0.05）"，不要只写"相关性显著"而不带具体水平。
- 相关系数旁的星号是显著性标记，不是相关强度标记——r=0.15但样本量很大时也可能三星，不能等同于"强相关"。

## 3. 拟合优度的合理区间与异常提示
- 描述统计中重点看 `skew`（偏度）与 `kurt`（峰度）：|skew|>2 或 |kurt|>7 提示该变量分布严重偏斜，建议在论文中说明是否做了对数变换或缩尾处理（对应 `clean.py` 的 `log_vars`/`winsorize_vars`）。
- 相关系数矩阵中若多对解释变量两两相关系数 |r|>0.8，是后续OLS/面板回归中多重共线性的预警信号，应在描述统计部分提前提示读者，呼应后文回归结果的VIF/共线性讨论。

## 4. 标准误类型是否说明清楚
- Pearson相关系数的p值基于正态性假设的t检验，若样本量较小（n<30）且变量明显非正态，应在脚注说明"相关系数显著性检验假设变量近似服从正态分布"，避免审稿人质疑检验有效性。

## 5. 审稿人常见追问点
- "描述统计表里的标准差是总体标准差还是样本标准差？"——本工具固定输出样本标准差（ddof=1），与Stata `summarize` 一致，写作时直接说明"标准差"无需额外注明，但若被追问应能答出依据。
- "相关系数矩阵能否替代回归分析？"——明确答：不能，相关分析未控制其他变量，只是初步描述性证据，正式假设检验需以回归结果为准；论文中应把相关分析定位为"初步关联性检验"，回归结果才是核心论证依据。
- "为什么没有报告Spearman相关系数？"——若变量非正态或为序数变量，需说明本工具仅提供Pearson相关，若审稿人特别要求Spearman需手动用其他工具补充并注明。
