一、研究背景与变量说明

本研究以 A 股上市公司为样本，考察企业规模、盈利能力、成长性、资产有形性、非债务税盾、资产周转率、国有股比例、独立董事比例及管理层持股对资本结构（资产负债率）的影响，检验权衡理论（Trade-off Theory）与优序融资理论（Pecking Order Theory）在中国情境下的适用性。

八、主要结论

  1. 盈利能力（ROA）对资本结构有稳健的负向影响（RE: β = −0.328，p<0.05；FE: β = −0.315，p<0.05），FE 与 RE 结果高度一致，结论稳健。支持优序融资理论——盈利企业优先使用内部留存收益，减少外部债务。[VERIFIED]
  2. 资产周转率（TURN）对资本结构有正向影响（RE: β = 0.079，p<0.1；FE: β = 0.095，p<0.05），说明资产营运效率高的企业具备更强偿债保障能力，可承载更高杠杆。[VERIFIED]
  3. 资产有形性（TANG）在混合 OLS 中高度显著为正（β = 0.711，p<0.01），RE 中边际显著（β = 0.294，p<0.1），FE 中不显著，说明有形资产的担保效应主要体现为横截面差异（不同企业间），而非时序变动效应。部分支持担保价值理论。[VERIFIED]
  4. 非债务税盾（NDTS）在混合 OLS 中高度显著为负（β = −6.669，p<0.01），RE/FE 中不显著，同样来源于企业间差异。OLS 结果支持替代效应假说。[INFERRED]
  5. 企业规模、成长性、国有股比例、独立董事比例、管理层持股在本样本中均未通过显著性检验。可能原因：样本量较小（N = 26 家公司），统计检验力不足；或样本期（2009–2016）中国资本市场的特殊环境所致。[INFERRED]
  6. 模型选择：Hausman 检验（χ² = 7.686，p = 0.566）不拒绝随机效应，但 ROA 和 TURN 的显著性结论在 FE 与 RE 中均一致，核心结论对模型选择具有稳健性。

稳健性提示：本研究样本量较小（N = 150），统计显著结论有限。建议后续研究扩大样本规模或延长时间窗口以提升检验力。
