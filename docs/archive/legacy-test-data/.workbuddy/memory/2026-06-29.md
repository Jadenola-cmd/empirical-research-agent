# 2026-06-29 工作日志

## 实证分析：资本结构影响因素

### 任务概述
用户提供了8个Excel数据文件（CG_Capchg、CG_Ybasic、FI_T1~T8、FS_Combas、FS_Comscfi），
要求对"资本结构的影响因素"做整体实证分析。

### 数据情况
- 原始样本：190条（26家公司 × 8年，2009–2016）
- 因变量：资产负债率 LEV（来自 FI_T1）
- 核心问题：FI_T1 中资产负债率有16条 > 1（最大58.08，股票代码688），
  原因是股东权益为负。解决方案：剔除 LEV > 1 的样本，剩余174条。

### 分析方法
按 empirical-research-agent skill 标准流水线执行：
1. 数据合并（8个文件按 code + date 合并）
2. 变量构建（LEV/SIZE/ROA/GROWTH/TANG/NDTS/TURN/STATE/INDEP/MSHARE_LN）
3. 缩尾处理（1%/99%）
4. 描述性统计 + 相关系数矩阵
5. 混合OLS回归（用户特别要求增加）
6. VIF多重共线性检验（全部<5，通过）
7. 面板回归（FE + RE，公司层面聚类标准误）
8. Hausman检验（linearmodels API问题未成功，以FE结果为主）

### 主要结果
- ROA（盈利能力）：FE中 β=-0.315，p<0.05，支持优序融资理论
- TURN（资产周转率）：FE中 β=0.095，p<0.05，显著为正
- TANG（资产有形性）：OLS中显著，FE中不显著（个体间差异）
- NDTS（非债务税盾）：OLS中显著为负，FE中不显著
- SIZE/STATE/INDEP/GROWTH：均未通过显著性检验（样本量N=162偏小）
