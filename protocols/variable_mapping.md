# Variable Mapping Protocol — 变量映射确认

## 位置

标准流水线 Step 4A。任何回归类方法执行前必须通过本协议。

## 必填字段

每个被纳入模型的变量必须列出：

| 字段 | 要求 |
|---|---|
| 角色 | Y / X / 控制变量 / 固定效应 / 聚类变量 |
| 来源表 | 原始数据表或合并后数据集名称 |
| 来源字段 | 原始字段名 |
| 计算方式 | 原始字段、比例、对数、差值、滞后项、标准化等 |
| 经济含义 | 变量在研究问题中的含义 |
| 方向说明 | 数值越大代表什么，转换后系数方向如何解释 |

## 重点说明

- Y 必须说明是创新投入、创新产出、创新质量，还是其他被解释对象。
- X 必须说明值越大代表税负越重、避税程度越高，还是其他经济方向。
- 变量经过对数、反向、差值、比例、标准化或 winsorize 等转换时，必须说明对系数方向的影响。

## 阻断条件

出现以下任一情况，不得执行回归：

- Y 或 X 未明确。
- 变量角色不清。
- 来源字段无法追溯。
- 计算方式无法复现。
- 变量方向可能反向但未说明。
- 用户尚未确认变量映射。

## 非交互降级

如果非交互环境无法等待用户确认，只能输出探索性报告，并在报告开头标注：

```text
[WARN] 未经过变量/样本确认，本报告仅为探索性结果，不得作为正式结论。
```
