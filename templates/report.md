# Report Template — 内部汇报格式

## 适用场景

课题组内部讨论、组会汇报、预答辩准备。

## 报告骨架

```
# {分析主题} — 实证分析报告

## 1. 数据概览

- 样本：{N}个观测值，{entity_count}个{entity_type}
- 期间：{start} - {end}
- 数据来源：{来源}

## 2. 核心发现

> **一句话结论**：{核心发现一句话}

## 3. 关键图表

### 描述性统计

{描述统计表 + 关键变量的分布图}

### 回归结果

{系数图（森林图），直观展示各变量系数和置信区间}

{三线表格式回归系数表}

## 4. 结果解读

{按可信度标签组织的要点列表}

### [VERIFIED] 可确认的结论
- {结论1}
- {结论2}

### [INFERRED] 可推断的结论
- {结论3}

### [SPECULATIVE] 待验证的推测
- {结论4}

## 5. 稳健性检验（如有）

{稳健性检验汇总表 + 综合评定}

## 6. 后续建议

- {后续分析方向1}
- {后续分析方向2}
```

## 格式要求

- 可视化优先：关键结果尽量配图
- 要点式解读：避免长段落，用要点列表
- 结论分层：用可信度标签区分确定性和推测性结论
- 行动导向：后续建议具体、可执行
