# Methodology Reference — 计量方法参考

## 标准误选择指南

| 数据类型 | 推荐标准误 | engine参数 |
|---------|----------|-----------|
| 截面OLS（无异方差） | 普通标准误 | se_type=null |
| 截面OLS（异方差） | HC1稳健标准误 | se_type=HC1 |
| 面板数据 | 聚类到个体 | se_type=clustered, cluster=entity |
| 小样本面板 | HC3 + 聚类 | se_type=HC3 |

## 面板模型选择

| Hausman检验结果 | 模型选择 | 标注 |
|---------------|---------|------|
| p < 0.05 | 固定效应（FE） | [VERIFIED] |
| p >= 0.05 | 随机效应（RE） | [VERIFIED] |
| chi2为负 | 保守选择FE | [INFERRED] |

## 中介效应判定

| 条件 | 判定 |
|------|------|
| a显著 + b显著 + c''不显著 | 完全中介 |
| a显著 + b显著 + c''显著且系数减小 | 部分中介 |
| a或b不显著 + Sobel显著 | 间接效应存在（[INFERRED]） |
| a或b不显著 + Sobel不显著 | 无中介效应 |

## 因果推断注意事项

### DID
- 必须检验平行趋势（事前趋势无显著差异）
- 处理组和对照组在政策前应有相似趋势
- 若平行趋势不成立 → 结果降级为 [SPECULATIVE]

### PSM
- 必须报告匹配前后平衡性检验
- 必须报告共同支撑域（common support）
- 卡尺宽度选择需要敏感性分析

### IV
- 第一阶段F统计量 > 10（弱工具变量检验）
- 必须报告过度识别检验（多工具变量时）
- 仅支持单一工具变量时为 [beta]

### RD
- 必须报告带宽选择的敏感性
- 必须报告McCrary操纵检验
- 多项式阶数选择需要敏感性分析
