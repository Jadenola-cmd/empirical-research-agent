# Reporting Protocol — 输出规范

## 位置

输出规范 [P4]：在 protocol_order.md 中位于 result_validation 之后，是最终输出前的最后一道关卡。

## 报告骨架

每次正式分析输出按以下章节组织（章节标题用中文数字编号，无内容的章节跳过）：

```
任务卡
  - 当前研究场景：研究探索 / 模型执行 / 结果迭代
  - 完成等级：探索性 / 草稿级 / 已验证
  - 是否经过变量确认：是 / 否
  - 是否经过样本构建确认：是 / 否
  - 是否可作为正式结论：是 / 否

一、数据概况
  - 数据来源、样本量、时间/个体维度（一两句话）
  - 变量定义表（仅列出本次分析实际使用的核心变量）

二、描述性统计
  - 三线表格式的描述统计表

三、相关性分析
  - 下三角相关系数矩阵（含显著性星号）

四、回归结果
  - 核心回归系数表（三线表格式）
  - 附加检验（Hausman / Sobel / 平行趋势等）

五、结果解读
  - 按 interpretation 清单生成的解读文字
  - 每条结论标注可信度标签 [VERIFIED]/[INFERRED]/[SPECULATIVE]

六、稳健性检验（如有）
  - 稳健性检验结果 + 解读
```

## 任务卡规则

- 每份报告开头必须先输出任务卡。
- 研究探索模式默认完成等级为“探索性”，是否可作为正式结论为“否”。
- 模型执行模式在完成变量确认和样本构建确认、且结果验证通过后，可标注为“已验证”；未完成任一确认时只能标注为“探索性”或“草稿级”。
- 结果迭代模式必须在任务卡后输出 `result_diff`。
- 如非交互环境无法完成变量/样本确认，任务卡后必须输出：

```text
[WARN] 未经过变量/样本确认，本报告仅为探索性结果，不得作为正式结论。
```

## 三线表标准

所有统计表格遵循学术三线表格式：

1. **顶线**：表头与内容之间的粗线
2. **中线**：表头内部的细线
3. **底线**：表格结束的粗线

### 描述统计表

| 变量 | N | 均值 | 标准差 | 最小值 | P25 | 中位数 | P75 | 最大值 |
|---|---|---|---|---|---|---|---|---|
| roa | 100 | 0.085 | 0.042 | -0.05 | 0.06 | 0.08 | 0.11 | 0.21 |

注：标准差为样本标准差（ddof=1）。

### 回归系数表

| 变量 | (1) roa | (2) roa |
|---|---|---|
| leverage | -0.532*** | -0.487*** |
|  | (-3.214) | (-2.891) |
| size | 0.118** | 0.105* |
|  | (2.087) | (1.952) |
| _cons | 0.062* | 0.071** |
|  | (1.823) | (2.104) |
| N | 100 | 100 |
| R² | 0.284 | 0.312 |
| F统计量 | 12.41*** | 14.52*** |

注：括号内为t值，robust(HC1)标准误，***p<0.01, **p<0.05, *p<0.1。

### 约定

- 系数和星号同行，t值在下行括号内
- 常数项 `_cons` 固定放在系数列表最后
- 模型统计量（N / R² / F）单独列在表格下方
- 标准误类型在表尾脚注中明确标注

## 变量定义表

| 变量符号 | 变量名称 | 定义与计算方式 |
|---|---|---|
| ROA | 总资产净利润率 | 净利润 / 总资产平均余额 |
| LEV | 资产负债率 | 总负债 / 总资产 |
| SIZE | 公司规模 | 总资产的自然对数 |

## 解读文字规范

- 每条结论标注可信度标签
- 解读中引用的数字必须与表中一致
- 统计量（p值、t值、R²）保留3位小数
- 避免"显著"一词滥用——仅当 p<0.05 时使用"统计显著"
- 经济显著性与统计显著性分开讨论

## 模板选择

根据用户场景选择输出模板：
- `templates/thesis.md` → 学位论文附录（完整严谨）
- `templates/paper.md` → 期刊投稿（简洁标准）
- `templates/report.md` → 内部汇报（可视化优先）

## Tab分隔复制块

默认不输出 Tab 分隔复制块，避免报告冗长并影响论文正文可读性。

仅当用户明确要求“给我可复制到 Excel/Word 的 Tab 表格”时，才在对应表格后追加 TSV 代码块。
